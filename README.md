# Toronto Traffic Collision Analytics Tool

## Project Overview
This project analyzes Toronto traffic collision data using Python and Streamlit.  
It provides insights into traffic patterns through interactive visualizations and filters.

---

## Sprint 1 Scope
- Load dataset
- Validate schema
- Clean missing neighbourhood values
- Remove invalid coordinates
- Analyze collisions by hour
- Analyze collisions by neighbourhood
- Generate hourly visualization
- Build a minimal Streamlit dashboard

---

## Sprint 2 Enhancements
- Analyze collision trends by year
- Analyze collision patterns by weekday and month
- Analyze road user involvement
- Add interactive filters (year and neighbourhood)
- Improve dashboard visualizations and layout

---

## Features
- Collision analysis by hour, weekday, month, and year
- Neighbourhood-based insights
- Road user involvement distribution
- Interactive filtering for deeper analysis
